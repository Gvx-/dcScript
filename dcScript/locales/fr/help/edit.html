Ce plugin permet d'ajouter du code HTML dans les behaviors des pages publiques du blog

Selection du behavoir a editer
La selection d'un behavior a editer ne peut se faire que si le behavior courant n'a pas été modifié
Actif
Le code sera inseré dans les pages publiques uniquement s'il est actif
Edition du code a inserer
L'editeur de code utilise la coloration syntaxique de CodeMirror. Voir egalement l'aide de la fonction recherche
Sauvegarde du code
Vous pouvez sauver en local votre code en utilisant le bouton Télécharger1
Accès a la configuration
L'acces à la configuration du plugin se fait par le lien Paramètres2

Notes:

1 - Disponible uniquement en absence de modification depuis le dernier enregistrement.

2 - Disponible uniquement pour les administrateurs.
