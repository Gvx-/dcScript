Le système de cryptage utilisé dans les versions précédentes est incompatible avec PHP 7.2 et plus.

Votre serveur est équipé d'une version PHP 7.2 ou plus.

Il vous faut donc utiliser cette page pour convertir vos scripts.

Procédure de conversion de cryptage

  1. Cliquer sur page de décryptage
  2. Cliquer sur Copier la clé
  3. Passer sur la page de décryptage
  4. Coller dans Clé de cryptage
  5. Revenir sur cette page
  6. Cliquer sur Copier le code du header
  7. Passer sur la page de décryptage
  8. Coller dans Element crypté
  9. Cliquer sur le bouton Décrypter
  10. Cliquer sur le bouton copier
  11. Revenir sur cette page
  12. Coller dans Code du header
  13. Cliquer sur Copier le code du footer
  14. Passer sur la page de décryptage
  15. Coller dans Element crypté
  16. Cliquer sur le bouton Décrypter
  17. Cliquer sur le bouton copier
  18. Revenir sur cette page
  19. Coller dans Code du footer
  20. Cliquer sur le bouton Convertir

Voila c'est terminer.

Ressources

Le fichier decryt.php est disponible dans le dossier dcScript/tools du plugin.

Ce fichier peux être déposé sur un serveur PHP avec PHP compris entre 5.3.3 <= PHP <7.2.

Ensuite vous pouvez utiliser ce fichier en lieu et place de la page extérieure mise à disposition.
