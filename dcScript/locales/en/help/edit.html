This plugin allows you to add HTML in the Behaviors blog public pages

Selecting the editing behavoir
The selection of a behavior has edit can be done if the current behavior has not changed
Active
The code will be ready inserted in the public pages only if it is Active
Edition of the code to insert
The code editor uses syntax highlighting CodeMirror. See also using the search function
Saving the code
You can save locally your code using the Download1
Access to configuration
The access to the plugin configuration is via the Settings2

Notes:

1 Available only in no change since the last save.

2 - Available only to administrators.
