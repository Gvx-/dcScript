This plugin allows you to add HTML code to the header (< head >) and footer blog public pages

Editing the code to insert
The code editor uses syntax highlighting CodeMirror. See also using the search function
Tab header code
Tab footer Code
Use this tab to edit the code that will be ready inserted in the footer of public pages
Saving the code
You can save locally your code using the Download*
Access to configuration
The access to the plugin configuration is via the Settings

* Available only in no change since the last save.
