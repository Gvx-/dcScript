The encryption system used in previous versions is incompatible with PHP 7.2 and higher.

Your server is equipped with a version of PHP 7.2 or higher.

So you need to use this page to convert your scripts.

Encryption conversion procedure

  1. Click on decryption page
  2. Click on Copy the key
  3. Go to the decryption page
  4. Paste in Encryption key
  5. Return to this page
  6. Click on Copy header code
  7. Go to the decryption page
  8. Paste into Encrypted element
  9. Click on the Decrypt button
  10. Click on the copy button
  11. Return to this page
  12. Paste into Header code
  13. Click on Copy the code of the footer
  14. Go to the decryption page
  15. Paste into Encrypted element
  16. Click on the Decrypt button
  17. Click on the copy button
  18. Return to this page
  19. Paste in Footer's code
  20. Click the Convert button

that's all

Resources

The decryt.php file is available in the dcScript/tools folder of the plugin.

This file can be uploaded to a PHP server with PHP between 5.3.3 <= PHP <7.2.

Then you can use this file instead of the external page provided.
